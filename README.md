# Dead Simple Password Generator

A dead simple password generator website that allows you to create secure passwords with customizable options.

## Features

- Adjustable password length with slider
- Generate multiple passwords at once
- Include or exclude uppercase letters
- Include or exclude lowercase letters
- Include or exclude numbers
- Include or exclude symbols
- Exclude similar characters (0,O,o,L,l,i,1)
- Exclude ambiguous symbols ({}[]()/\'"~,;:.<>)
- Copy individual passwords
- Copy all passwords at once

## Usage

1. Open index.html in any web browser
2. Adjust the password length slider and amount
3. Select your character preferences
4. Click "Generate Password"
5. Copy passwords using the copy buttons
